# Client thing

## Setup

 1. Install Node v8
 2. `npm install`
 3. `npm install` in `/visualizer` en `/blockly` en `/gameserver`
 3. `npm install webpack` or `(sudo) npm install webpack -g`
 4. Run `webpack` in `/client`
 5. Run `node server.js` in `/gameserver`
 6. The magic happens at `localhost:3000`
